## 贡献方法

先创建一个相应的分支，增删改相应的内容，然后发 PR 到 master，其中 reviewer 至少圈一个人。比如： @huchengquan, @anchengjian, @wangzhen 等。

## dev

``` bash
# 本地预览
# 先安装 http://www.mkdocs.org/

mkdocs serve

# build
mkdocs build
```
